Exercise 2: Finding Unknown (5 marks)

Given that:

2x14
31-21
=
42+x-55

Find the value of x

Solution:

Multiply the matrices on the left side:

(1,1): 2(3) + x(-2) = 6 - 2x

(1,2): 2(1) + x(1) = 2 + x

From (1,1): 6 - 2x = 4

-2x = -2

x = 1

Verify with (1,2): 2 + x = 2 + 1 = 3 ≠ 2+x from RHS

If RHS (1,2) should be 2+x, then x = 1 is correct!

Exercise 4: Identity Matrix (4 marks)

Find values of a and b such that:

3254
ab-53
=
1001

(This means the second matrix is the inverse of the first)

Solution:

Multiply and set equal to identity matrix:

From (1,1): 3a + 2(-5) = 1

3a - 10 = 1

3a = 11

a = 11/3

From (1,2): 3b + 2(3) = 0

3b + 6 = 0

3b = -6

b = -2

Verify: Check (2,1) and (2,2) to confirm!
